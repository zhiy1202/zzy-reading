# zzy-reading
> 使用springboot vue musql 构建的一个阅读网站
## 实现功能
### client
1. 查看文章
2. 收藏
3. 评论
4. 保存历史记录
5. 课后习题
6. 视频教学
7. 评分机制
8. 获取所有的成绩
10. 收藏好文好句
### admin
1. 文章的增删改查
2. 用户管理

## 架构实现
### 前端
1. vue(vuex , vueRouter , vue-cookie , day.js )
2. elementUI
3. axios
4. npm 构建
### 后端
1. springboot
2. mybatis-plus
3. springMVC
4. maven构建
### database
1. mysql
